FEATURE ENGINEERING

Feature Selection

The general pipeline of feature selection includes several steps:

  1. searching

    a. Complete search(Breadth First Search, Branch and Bound, Beam Search, and Best First Search)

    ​ I. Exhaustic

    ​ II. non-exhaustive

    b. heuristic search(Sequential Forward Selection (SFS), Sequential Backward Selection (SBS), and Bidirectional Search (BS)). SFS and SBS iteratively delete features from a full set or add features to an empty set.

    c. random search(Simulated Annealing (SA) and Genetic Algorithms (GA))

  2. Evaluation

    1. Filter method. Scores each feature according to correlation or divergence and then select by threshold. The scoring commonly employs variance, correlation coefficient, Chisquare test, and mutual information as criteria.
    2. Wrapper method. Perform classification on a sample dataset with selected label and use accuracy or sth like that as criteria.
    3. Embedded method. Perform variable selection as part of learning procedure like Regularization, decision tree, and deep learning .

MODEL GENERATION

A. Model Structures

1)Entire structure

search the entire network from scratch. Search space is huge and cannot share the same structure for both small and huge dataset.

2) Cell-based structure

search space is much smaller than the above one. For large dataset, simply stach more layers.

3) Hierarchical Structure

The low level structure is the base for generating the upper level structure.

4) Network Morphism based structure

is the way of transferring the information stored in an existing neural network into a new neural network.

B. hyperparameter optimization(HPO)

1) Grid & Random Search

  1. grid search: Traditional grid search wastes a lot of computation resource because not all parameters are important. Modified grid search: search in a coarse area, and then perform a finer grid search.

  2. random search: it is hard to determine whether the found parameters is the best set.

2) Reinforcement Learning
3) Evolutionary Algorithm
4) Bayesian Optimization

build a probability model of the objective function and employ the model to select the most promising hyperparameters. Finally evaluating the choosen hyperparameters on the true objective function.

5) Gradient Descent

Model Estimation

model evaluation is expensive. Therefore different kinds of acceleration for model evaluation is proposed.

A. Low fidelity

Use modified dataset(less examples, smaller model)

B. Transfer learning

share parameters between child networks or inherit weights from previous design.

C. Surrogate

try to approximate the original expensive objective rather than directly evaluation.

D. Early stopping
